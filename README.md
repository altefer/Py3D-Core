# Py3D-Core
简单易用的Python 3D游戏引擎，基于Pygame和PyOpenGL



## 🎉 Py3D-Core v1.0.0 - 首个正式版本

### ✨ 特性

- **简单易用**: 无需复杂的配置，几行代码即可创建3D场景
- **基于Pygame**: 利用Pygame进行窗口管理和输入处理
- **OpenGL渲染**: 使用PyOpenGL进行高性能3D渲染
- **组件化架构**: GameObject-Component设计模式，易于扩展
- **内置示例**: 包含多个游戏示例，开箱即用

### 📦 安装

```bash
pip install Py3D-Core
```

### 🎮 快速开始

```python
from py3d.core.game_object import GameObject
from py3d.core.components.mesh_renderer import MeshRenderer
# 更多示例参见 examples/ 目录
```
